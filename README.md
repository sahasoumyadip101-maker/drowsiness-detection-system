# 🚗 Driver Drowsiness Detection System

## Overview
This project detects driver drowsiness and yawning in real-time using computer vision and machine learning.

## Features
- Eye detection using CNN
- Yawning detection using CNN + MediaPipe
- Real-time webcam monitoring
- Alarm alert system
- Email alert (disabled for security)

## Technologies
- Python
- OpenCV
- TensorFlow
- YOLOv8
- MediaPipe

## How to Run
pip install -r requirements.txt  
python drowsiness_cnn.py
